# Java Maze Solver

### Created by Jason Domogala and Benjamin Whitehead

To compile the MazeSolver program:

```shell
javac MazeSolver.java
```

To run MazeSolver:

```shell
java MazeSolver
```

To compile the MazeGenerator program:

```shell
javac MazeGenerator.java
```

To run MazeGenerator

```shell
java MazeGenerator
```
